# 萌萌 Clawd 的日常 🦀

![萌萌 Clawd 的日常横幅](wechat/01-banner-750x400.png)

一套记录桌面小螃蟹 Clawd 日常情绪的动态表情包。

Clawd 会陪你工作、替你加油，也会大叫、尴尬、无语和当场晕倒。虽然它的钳子只有两个小方块，但表达感情非常认真——尤其是在面对喜欢的人时。

## 表情一览

| 文件名 | 表情 |
| --- | --- |
| `chi-wo-yi-quan` | 吃我一拳 |
| `da-jiao` | 大叫 |
| `gan-ga` | 尴尬 |
| `hao` | 好 |
| `lao-po` | 老婆 |
| `wu-yu` | 无语 |
| `yun` | 晕 |
| `jia-you` | 加油 |
| `ni-xing-la` | 你醒啦 |

浏览器可直接打开 [`gallery/index.html`](gallery/index.html) 查看九宫格动画图鉴。

## 文件结构

```text
emotes/          透明底 GIF 与可编辑动画 SVG
gallery/         九宫格 HTML 图鉴
preview/         图鉴预览图
wechat/          微信表情开放平台所需横幅、封面和聊天图标
```

## 内容提醒

本系列含有轻微的**人机恋、电子陪伴**及“称呼人类为老婆”等虚构情感互动要素，请根据个人喜好观看。
